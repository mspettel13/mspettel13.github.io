Notation:

  • AxB = B sets of A reps
  • FxB = B sets till failure
  • AxBxC = C sets of A reps of one exercise and B reps of another exercise with no rest between the two exercises
  • Drop Set = A set where you start around your normal weight and each "set" you fail you decrease the total weight by 10-20lbs
  • Super Set = Doing a set of two different exercises back to back with no rest
  • Negative = You start at the top of the rep and lower the weight as slowly as possible (feel the burnnnn)

Goals and Rep Counts:

  • The rep and set suggestions above are meant to strike a balance between getting stronger and getting bigger. If you want to focus on getting stronger, aim for 4-6 reps per set (and lift really heavy). If you want to focus on getting bigger, aim for 8-12 reps per set.
